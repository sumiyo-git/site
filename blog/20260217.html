Chromium

记录 mio 在研究 Chromium 时遇到的坑 ... 本文讨论的是纯正的 Chromium-browser，而不是 “基于 Chromium 开发的浏览器” 哦。 Chrome 自动更新、鼠标左键默认在当前标签页打开书签、不支持鼠标手势和双击关闭标签页等功能，确实有点让人失望。既然保证不了方便快捷，那至少得保证安全隐私吧？

所以 mio 把目光转向了 Chromium 浏览器。与 Chrome 不同，Chromium 是开源的，没有集成谷歌服务，自然更注重隐私。结果一番测试下来，发现这玩意儿根本不适合在日常生活中使用。呜呜，mio 还是去找找其他浏览器吧 ...

实机测试结果，放这里供大家参考哦。

mio 不建议使用 ungoogled-chromium 或其他由第三方编译的 chromium 浏览器，谁知道他们在编译之前加了些什么进去呢。

通过 chrome://gpu/ 可以看到，Chromium 缺少 h264 等专利编解码器，因此无法播放大多数视频。 这已经远超 mio 的能力范围了呐

每次启动弹出的 “缺少 Google API 密钥，因此 Chromium 的部分功能无法使用。” 提示，可以通过添加以下加以下的环境变量来隐藏掉哦。
变量名 变量值
GOOGLE_API_KEY no
GOOGLE_DEFAULT_CLIENT_ID no
GOOGLE_DEFAULT_CLIENT_SECRET no
