其他

来源注明
在发布来自本站的内容时，您需要注明来源。完整的注明应包括以下信息： 作品名（可选） 作者（建议填入 mio） 来源（建议填入网址） 例如："アビス信仰" by mio, from: https://sumiyo.link/?id=20231002

盈利行为
我们所禁止的盈利行为包括但不限于 直接获利（售卖） 和 间接获利（三连获取资源、入群获取资源） 等形式

禁止未授权的使用
sumiyo-git 在 github 上开放了此网站的所有代码。这些代码仅可用于学习和公众审查，禁止一切未授权的使用，包括但不限于自行部署、分发、二次发布

留言板

特性
超过 5 行的留言和超过 3 行的回复将会被折叠 留言中，匹配 http(s)://*/ 格式的内容将被替换为可点击的链接 1 条留言下最多只能有 6 个回复 超过 7 天的留言 & 回复将无法被删除 备注: 在未来相当长的一段时间内，mio 不打算添加 直接显示图片、内嵌 iframe、表情包、Markdown 语法、点赞系统 等功能
注意事项
mio 会不定期检查留言数据的哦，请不要发表过激言论 后端是 mio 自己搭的，有被黑客拍扁的可能。请不要写入个人敏感信息 留言昵称可复用，请自行斟酌留言内容。同样，mio 不会为您因使用留言板服务所造成的任何后果负责哦

后台日志
您的每个请求都会被 Cloudflare 记录。mio 也会不定期进行抽样检查。以下为全部参数: 请求发起时间 请求方式 自治系统编号（ASN） 自治系统编号详细信息字段 IP 地址 地理位置（精确到国家） 请求路径 浏览器 UserAgent

Cookie 🍪
网站共使用 1 个 Cookie，数据以 json 的形式存储，有效期 15 天，有效期内的再次访问会自动续期。这里是已被占用的键值列表：
键值 作用
date 记录该 Cookie 生成 / 续期的时间
is_new 用于网站访问量统计
theme 记录当前深色模式开关


留言板
使用留言板服务会额外记录您的以下信息: 留言发表时间 留言昵称 留言内容

访问量统计
这是 mio 自己搭的访问量统计 api 哦，不会额外收集任何用户信息，也不受第三方平台干预。

数据存储
留言板、访问量等数据储存在 Cloudflare D1 数据库中，另有少部分日志文件放在 mio 自己的电脑上。
